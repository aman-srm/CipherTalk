🔐 CipherTalk — Secure Messaging
CipherTalk is a browser-based end-to-end encrypted messaging app. Every message is encrypted directly in your browser before it reaches the server — meaning no one, not even the server, can read your conversations.

🌐 Live App: https://cipher-talk-phi.vercel.app/

💬 What is CipherTalk?
CipherTalk lets you chat privately with friends using military-grade encryption. You register with a username and passphrase, add friends, and start messaging — all without worrying about your data being exposed. Files, photos, and videos are encrypted too.

✨ Key Features

🔒 End-to-end encryption — messages encrypted with ECDH + AES-GCM-256
📁 Encrypted file sharing — send photos, videos, audio, and documents
👥 Friends system — search users by username and send friend requests
⏱️ Message retention — View Once, 24 Hours, or Keep forever
🔓 Decrypt sessions — enter passphrase once, all messages auto-decrypt for 30 seconds
📱 Mobile friendly — fully responsive on all screen sizes
⚡ Real-time messaging — messages appear instantly without refreshing
🌍 Cross-device login — log in from any device with the same credentials
🔑 Tab lock — app locks automatically when you switch tabs
